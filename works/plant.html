[8]  RE-LEAF
2021
installation / device
1 plant, 1 screen, Arduino board, wires
// labour // human-machine // technology // interactive

//Words: 0

//Links: 0

Description
This project aims to research organic materiality and form and the human interaction with it. Through re- shaping our current office (man-made) environment and translating organic features onto non-organic ones, I want to explore the possibilities for sustainable design, but moreover evoke certain feelings we only experience when interacting with nature. Furthermore the installation aims to reshape our current approach to technology and comment on what our “natural“ behaviour came to be.

* using a plant as a capacitive switch, connected through Arduino Leonardo. By touching 4 different leaves, the user can operate the mouse cursor on the screen.
